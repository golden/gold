
- lower case: atom
- upper case: lists

|Symbol| Meaning|
|------|--------|
| n    | number                           |
| s    | sting                            |
| x    | string or number                 |
| V    | vector: a list with all numeric indexes |
| S    | a set: all indexes, no values    |
| A    | an array. Includes V and S       | 
| +    | an array, to updates             |
| -    | an empty array, to be filled in  |
| ?    | optional argument                |
