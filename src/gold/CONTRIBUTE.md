### Coding standards

|symbol| notes |
|------|-------|
| x    | any atom number or string |
| $    | number |
| "    | string |
| ?    | boolean (0.1), empty sting or string |
| +    | list to be augmented |
| -    | list to be replaced |
| v    | vector = array with numeric indexes |
| s    | set = array with sting indexes indexes |
| a    | array = either of v or s |
| n    | number |
